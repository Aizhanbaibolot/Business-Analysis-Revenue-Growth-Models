# **Business Analysis Revenue Growth Models**
### Project Overview <p>
The purpose of this  project was to minimize cost and maximize revenue of the Lariat Rent-A-Car Company for the upcoming year by using present year's data as a baseline and building different strategies to reach the goal. 
## Business objective

* Recommend Strategies to Reach the Compny's Goal
